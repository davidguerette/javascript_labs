Arrays in both JS and Ruby:

  • Can be instantiated instatiated with []
  • Zero-indexed
  • Can access an item with arrayname[indexnumber]

Declaring an array with values

  • Ruby: Array.new('Lannister','Baratheon','Stark')
  • Javascript: new Array('Lannister','Baratheon','Stark')

Appending Values to end of array

  • Ruby: <<, .push(value)
  • Javascript: .push(value)

Iteration

    var familyNames = ['Lannister','Baratheon','Stark'];

    for(var i = 0; i < familyNames.length; i++){
      console.log(familyNames[i]);
    }
